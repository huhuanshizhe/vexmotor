# Downloads / `/account/downloads`

## 1. 目的：集中下载已购产品文档；统一管理 CAD / 手册版本更新。

## 模块
- 过滤：产品 / 文档类型 / 语言 / 更新时间。
- 表：File / Type / Version / Language / Size / Updated / Download。
- 顶部 banner：有更新的文档数 → `Show updated only`。

## 交互：批量打包 ZIP；选定 SKU 一键打全套。
## SEO：noindex。
## 埋点：`downloads_bulk {sku, count}`。
## 验收：仅显示用户公司有过购买/询盘的 SKU 文档。
