# Contact / `/support/contact`

## 模块
- 两栏：左 Form / 右 Info（offices、电话、邮箱、工作时间、紧急工程支持电话）。
- Form：Topic（Sales / Technical / Order issue / Logistics / Press / Partnership）→ 动态字段；上传可选；公司 / 国家 / 邮件 / 电话 / 信息。
- 提交：成功页 + 工单 #，承诺响应时间。

## SEO：LocalBusiness × N（每个 office）。
## 埋点：`contact_submit {topic}`。
## 验收：Topic=Technical 时自动建议 FAQ 与 Selector。
